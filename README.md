## Сценарии для демо

### Успешный сбор

| | ИИН | ФИО |
|---|-----|-----|
| Автор (регистрация) | 880420301999 | Асель Бекенова |
| Получатель | 850315301234 | Айгуль Смагулова |

Низкий риск, есть в медреестре, есть диагноз. «Проверить» заполняет форму, сбор создаётся.

### Блокировка антифродом (есть в медбазе)

| | ИИН | ФИО |
|---|-----|-----|
| Автор | 880420301999 | Асель Бекенова |
| Получатель | 990101300999 | Ерболат Мукашев |

«Проверить» находит человека в медреестре, но сохранение сбора блокируется.

Альтернатива: получатель `880101300888` (Алмаз Токтаров).

### Блокировка регистрации автора

Регистрация с ролью **author** и ИИН `990101300999` → ошибка: «Регистрация невозможна: высокий уровень риска.»

Тот же ИИН с ролью **donor** → регистрация проходит.
